Časté dotazy

Co se vás budou ptát kolegové

Co znamená „dedikovaný server" v praxi?
Hardware (CPU, RAM, disky, síťová karta) patří výhradně jedné službě — vám. Žádný hypervisor, žádné sousední VM, žádný „noisy neighbor" efekt. Předvídatelný výkon a možnost spustit cokoli (vlastní hypervisor, kontejnery, custom kernel).
Jaké operační systémy podporujete?
Standardně Debian 12, Ubuntu 24.04 LTS, Rocky Linux 9, AlmaLinux 9, Proxmox VE 8. Reinstalace samoobslužně z portálu, ~5 minut. Pro custom OS použijete IPMI konzoli — bez omezení.
Jak funguje SLA 99,9 %?
Měsíční výpočet. 99,9 % = max 43 minut nedostupnosti. Při překročení automaticky kompenzujeme 5–25 % měsíční ceny formou kreditu. Plné znění SLA před objednávkou.
Můžu mít víc IPv4 adres?
Konfigurace DS-PRO a vyšší obsahují 2-4 IPv4 v ceně. Další lze doobjednat (40 Kč/měs/IP). IPv6 vždy alespoň /64, větší konfigurace /48.
Co když chci službu zrušit?
Zrušíte z portálu jedním klikem. Server běží do konce zaplaceného období, pak se deaktivuje. Žádné výpovědní lhůty, žádný lock-in. Zálohujte data včas — po deaktivaci se disky bezpečně mažou.
Reboot serveru o víkendu?
Reboot, reinstalace, IPMI konzole — vše samoobslužně z portálu, 24/7. Pro havarijní situace je k dispozici český support tým s první odpovědí pod 15 minut.
Posíláte zálohy?
Pravidelné zálohy nejsou součástí standardního pronájmu — server je váš, zálohy si řídíte. Pro klienty s requirements na managed backup máme storage konfigurace s NVMe cache + RAID 10. Pro off-site backup se domluvíme.
Můžu si server před objednávkou vyzkoušet?
Pro DS-PRO a vyšší konfigurace nabízíme 14denní zkušební období s plnou refundací bez udání důvodu. Stačí napsat na hello@nordhost.cz.
